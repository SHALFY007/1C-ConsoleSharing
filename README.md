# Console Sharing  🎮
Автоматизированная система управления бизнесом по аренде игровых консолей. Проект разработан для автоматизации ключевых бизнес-процессов: от учета оборудования до генерации договоров аренды.

🚀 Основные возможности
Управление парком техники: Полный цикл учета консолей, включая отслеживание серийных номеров.
Автоматизация выдачи: Удобный интерфейс оформления документов выдачи и возврата оборудования.
Генерация документов: Автоматическое формирование и печать договоров аренды прямо из документа.
Тарифная сетка: Гибкое управление ценами на аренду.
Учет взаиморасчетов: Мониторинг оплат и контроль дебиторской задолженности клиентов.

🛠 Технологический стек
Платформа: 1С:Предприятие 8.3
Язык разработки: BSL (Built-in Scripting Language)
Архитектура: Управляемые формы, клиент-серверное взаимодействие

📋 Структура конфигурации
Система разделена на логические подсистемы для удобства работы пользователя:
Аренда — ключевые операции по движению техники.
Склад — номенклатурный учет и серийные номера.
Финансы — управление движением денежных средств.
Аналитика — отчеты для контроля эффективности бизнеса.

🏗 Как развернуть проект
Установите платформу 1С:Предприятие 8.3.
Создайте новую информационную базу в режиме «Создание информационной базы из шаблона» или «Добавление существующей».
Используйте файл конфигурации console.cf.
